# 大转盘抽签系统

营业部晨会现场抽签工具：主持人电脑接大屏投影，随机抽取员工参与情景演练。单页 HTML 应用，零依赖、离线可用、数据本地持久化。

## 功能特性

- **员工管理**：增删改姓名（上限 50 人），重名自动拦截
- **三级状态机**：空闲 → 已上盘 → 已抽中，点击上盘、再点下盘
- **抽签动画**：Canvas 2D 绘制转盘，3 秒缓动旋转，5~8 圈随机停止，扇区高亮闪烁
- **重置 vs 刷新**：重置保留历史记录重抽；刷新清空历史开新一轮
- **历史记录**：倒序展示，关闭浏览器不丢失
- **持久化**：`localStorage` 存储，重启电脑数据仍在
- **响应式**：横屏（左转盘右面板）/ 竖屏（上转盘下面板）自适应
- **零网络依赖**：纯前端，无 CDN、无后端、无任何外部请求

## 快速开始

```bash
# 方式 1：直接双击 index.html 用浏览器打开
# 方式 2：命令行启动
start index.html        # Windows
open index.html         # macOS
xdg-open index.html     # Linux
```

首次打开会自动生成 12 个空白槽位，点击空白按钮输入姓名即可开始。

## 使用流程

1. 点击空白按钮 → 输入员工姓名（重名会被拦截）
2. 点击已命名的按钮 → 名字加入转盘（按钮变灰）
3. 再次点击灰色按钮 → 下盘（按钮恢复彩色）
4. 点击转盘中心「开始」按钮 → 旋转 3 秒后停止，右侧显示中签者
5. 中签者按钮变深灰禁用，历史记录自动追加
6. 顶部「重置」重抽本轮 / 「刷新」开启新一轮

## 技术栈

| 项目 | 选型 |
|---|---|
| 标记 | 原生 HTML5 |
| 样式 | 原生 CSS3（含 `backdrop-filter` 毛玻璃、CSS 动画） |
| 脚本 | Vanilla JavaScript (ES6) |
| 存储 | `localStorage`（键名 `wheelData`） |
| 绘图 | Canvas 2D API |
| 依赖 | 无（不引入任何 CDN / 第三方库 / 框架） |

## 浏览器兼容

Chrome 90+ / Edge 90+ / Safari 14+ / Firefox 88+

## 数据安全

- 所有数据仅存储在浏览器 `localStorage`，不联网、不上传
- 清除浏览器缓存会丢失员工名单与历史记录（建议定期手动备份名单）
- 不同浏览器、不同电脑的数据相互独立

## 设计理念

- **金融极简风**：深邃蓝黑 + 暖金质感，禁用绿色系
- **悬念感**：先快后慢的缓动曲线（`cubic-bezier(0.17, 0.67, 0.12, 0.99)`）
- **大屏友好**：1920×1080 投影适配，字号自动放大
- **小白可用**：双击即运行，无安装、无配置
